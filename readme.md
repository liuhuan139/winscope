###优化winscope的启动方式---当前在ubuntu下运行

##### 1. 运行install.sh
```
   chmod +x install.sh
   ./install.sh
```

##### 2. 在启动台会生成winscope的图标，点击运行
##### 3. 在启动的窗口上一次点击 start winscope和 start proxy，启动正常的情况下，点击 打开页面
##### 4. 第一次启动，需要将窗口中的proxy token 复制到开启的网页中！
##### 5. 为了正常连接android设备，请保证设备安装了adb 
```
    sudo apt-get install adb
```
